## Instructions

### Requirements
* PHP 8.1 or higher
* Composer
* Node.js

### Setup
* composer update
* php artisan migrate
* php artisan db:seed
* npm run dev
* php artisan serve

### Assignment
* Replicate this page: https://www.intellezy.com/blog
  * Include header and footer
* Use the branding guide: https://www.intellezy.com/branding
